== Description


A clone of the yelp website done in Rails. A user can sign_up create a review for a restaurant with rating. 

== Technologies Used


* RoR
* AJAX
* Capybara for feature testing

== How to use


* git clone git@github.com:fitstek/week8_yelp_rails.git
* cd week8_yelp_rails

* unfortunately you can only inspect the code with an editor as the secret.yml file has been ingnored because it contains some keys that can't be made public.
